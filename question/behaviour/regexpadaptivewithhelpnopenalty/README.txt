-------------------------------
How to install on a moodle site
-------------------------------
If you downloaded this zip archive from the new moodle.org plugins page
1.- Unzip the zip archive to your local computer.
2.- This will give you a folder named "regexpadaptivewithhelpnopenalty".
3.- GO TO STEP 4 below
---
If you downloaded this zip archive from https://github.com/rezeau
1.- Unzip the zip archive to your local computer.
2.- This will give you a folder named something like "rezeau-moodle-qbehaviour_regexpadaptivewithhelpnopenalty-e9dc3f4".
The end of the name may vary.
3.- ***Rename*** that folder to "regexpadaptivewithhelpnopenalty".
---
4.- Upload that regexpadaptivewithhelp folder to <your moodle>/question/behaviour/ folder.
5.- There is no need to Visit your Admin/Notifications page because that question behaviour do not need to be "installed"
(it does not add any fields to your moodle database).

6.- However, just to make sure you have installed the behaviour in its correct folder, you may visit your moodle Site administration
 / Plugins / Question behaviours / Manage question behaviours page, where you will see the new behaviour listed.

----------
HOW TO USE
----------

This plugin is one of the 2 question behaviours to be used exclusively with the regexp question type.
They enable you to take advantage of the *Help* feature, which allows the student to "buy" a letter.

Online Documentation at http://docs.moodle.org/25/en/Regular_Expression_Short-Answer_question_type.
