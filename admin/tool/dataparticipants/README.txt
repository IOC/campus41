Tool data participants plugin for Moodle
=============================================

This plugin allows managers to send a list of participants to an email using a scheduled task.

Project page: http://github.com/IOC/moodle-tool_dataparticipants

Authors
-------

Marc Català <mcatalal@ioc.cat>

Installation
------------

Unpack archive inside /path/to/moodle/admin/tool/dataparticipants

For general instructions on installing plugins see:
http://docs.moodle.org/35/en/Installing_plugins
