Local batch plugin for Moodle
============================

This plugin allows admin and category managers to batch processes like create new courses from one template, remove courses, restart courses and import Moodle 1.9 backups.

Project page: http://github.com/IOC/moodle-local-batch

Authors
-------

Marc Català <reskit@gmail.com>
Albert Gasset <albert.gasset@gmail.com>

Installation
------------

Unpack archive inside /path/to/moodle/local/batch

For general instructions on installing plugins see:
http://docs.moodle.org/24/en/Installing_plugins
