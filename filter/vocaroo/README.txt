VOCAROO FILTER PLUGIN
============================

Vocaroo is a Moodle 3.X filter that replace any Vocaroo link with an embedded player

Web: https://github.com/IOC/moodle-filter_vocaroo


Install
=======

Download code from GitHub, and copy inside directory MOODLE_PATH/filter/vocaroo
